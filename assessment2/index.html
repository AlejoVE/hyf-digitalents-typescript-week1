How does it work?

Caesar’s cipher uses an encryption algorithm where each letter is replaced with a letter by some fixed number of positions down the alphabetical series (e.g., a shift of 1 replaces the letter a with b, the letter e with f, the word hello with ifmmp, etc.).

  • Encryption happens when the “shift” value is set.
  • Encryption of the text is updated which each new character typed by you given there is a value set for “shift”.
